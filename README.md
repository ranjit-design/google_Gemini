# Google Gemini - AI-Powered Assistant

Google Gemini is an AI-powered assistant built using React and Vite. It leverages Google's Generative AI to provide intelligent responses to user prompts. The application features a clean and interactive UI with components like a sidebar for navigation and a main area for user interactions.

## Features

- **AI-Powered Chat**: Communicate with Google's Generative AI for intelligent responses.
- **Recent Prompts**: View and reuse previously entered prompts.
- **Interactive UI**: A responsive and user-friendly interface.
- **Customizable Settings**: Options for managing activity, help, and settings.

## Technologies Used

- **React**: For building the user interface.
- **Vite**: For fast development and build processes.
- **Google Generative AI**: For AI-driven responses.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server with `npm run dev`.
